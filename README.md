# Data Visualistion and logging of mood
This is a code i developed to track mood over the course of the last 30 days and visualise the data into a graph

# Mental Health Mood Tracker
A Python CLI tool to log, analyse, and visualise your daily mental wellbeing

# Features
Log moods daily (scored 1–5 with emoji labels)
Tag emotions from a curated list (Anxious, Calm, Stressed, etc.)
Add notes to each entry
Summary statistics: average mood, best/worst days, top emotions
Auto-generated charts, mood trend over time + emotion frequency bar chart
Data stored locally in mood_log.json

# Tech Stack
Python 3
Pandas (data handling)
Matplotlib (visualisation)
JSON (persistent local storage)

# How to Run
bash
pip install pandas matplotlib
python mood_tracker.py
Why I Built This
As a Psychology graduate with a background in healthcare and data analysis, I wanted to combine behavioural science principles with practical Python skills. This project bridges both worlds — mental health awareness and data engineering.
